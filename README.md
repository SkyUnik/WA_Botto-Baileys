## Getting Started

This project require NodeJS v12.
Recommended NodeJS v16~.

### Install
Clone this project

```bash
> git clone https://github.com/SkyUnik/WA_Botto-Baileys.git
> cd WA_Botto-Baileys
```

Install :

```bash
> npm install 
```

### Usage
Run the Whatsapp baileys

```bash
> npm start
```
after running it you need to scan the qr / spesify the auth .json file
